## ‚úçÔ∏è Content Writing Guidelines

### Article Structure (2000-2500 words)
```
1. Title (H1) - Include primary keyword
2. Meta Description (155 chars) - Include keyword + CTA
3. Introduction (150-200 words)
   - Hook with problem statement
   - Promise solution
   - Include tool CTA

4. Table of Contents (for posts >1500 words)

5. Main Content Sections (H2)
   - What is [Tool]? (300 words)
   - How to Use [Tool] (400 words)
   - Benefits & Features (300 words)
   - Step-by-Step Tutorial (500 words)
   - Tips & Best Practices (300 words)
   - Common Mistakes to Avoid (200 words)

6. FAQ Section (5-8 questions)
   - Target "People Also Ask"
   - Use FAQ schema

7. Conclusion (150 words)
   - Summarize key points
   - Strong CTA to tool

8. Related Tools Section
   - Link to 3-5 related tools
   - Link to 2-3 related blog posts
```

### SEO Optimization Checklist
- [ ] Primary keyword in title, H1, first paragraph
- [ ] Secondary keywords in H2 headings
- [ ] LSI keywords naturally throughout
- [ ] Internal links to 5+ tools
- [ ] Internal links to 2+ blog posts
- [ ] External links to 2-3 authority sites
- [ ] Images with descriptive alt text
- [ ] Meta description with keyword
- [ ] URL slug with primary keyword
- [ ] Schema markup (Article + FAQ)

### Content Quality Standards
- **Readability:** Grade 8-10 (Hemingway App)
- **Uniqueness:** 100% original (Copyscape)
- **Keyword Density:** 1-2% for primary keyword
- **Images:** Minimum 3 custom images/screenshots
- **Examples:** At least 2 real-world examples
- **Tool Embed:** Embed actual tool when possible
